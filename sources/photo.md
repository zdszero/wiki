## 构图

打开 **九宫格**

```
         col1   col2  col3
        ___________________
        |     |     |     |
 row1   |  1  |  2  |  3  |
        |_____|_____|_____|
        |     |     |     |
 row2   |  4  |  5  |  6  |
        |_____|_____|_____|
        |     |     |     |
 row3   |  7  |  8  |  9  |
        |_____|_____|_____|
```

### 全身照

* 显腿长构图法

身体占据row2和row3，row1留给其他空间

* 1/2构图法

把人放在左边或者右边的1/2处，适合靠着某些东西（比如电话亭或者一棵树）

### 半身照

不要干巴巴地对着脸拍，不是拍证件照，没必要那么正式

用下方的左四格或者右四格，把头上的天空，身边的景色都露出来
